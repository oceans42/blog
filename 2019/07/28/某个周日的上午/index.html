某个周日的上午

Posted on 2019-07-28

我的第一个产品

昨天做了个小工具，叫锦囊妙计，目前只有一个页面，已经放到网上了，我会先自己用，把它逐步完善到自己满意。再给别人用（如果有人肯用的话）

做这个东西的缘由是，我觉得有时人会比较清醒，有时会比较迷惘、忧虑。有时灵光一现，有绝妙的想法，但可惜一闪而过，并没有记录下来，就算记录在日记或笔记里，日后需要时也很难在一堆东西里找到，所以一直想要一个纯粹的记录我最重要灵感的地方。可能一句话，就能点醒此刻的无明。而日记、笔记都不太适合来存放。所以，它诞生了。

其实，此刻的我正感到烦闷、忧虑。我会搜罗一些有帮助、智慧的句子，存放到锦囊里，下次再出现类似的情绪，就不怕了。

比如，很多事情一开始，其实我们是看得清的，但是随着后续发展，我们会陷入迷茫，就像《阿凡达》，其实杰克一开始录视频日记时就说了，它们绝对不可能放弃自己的家园潘多拉。但是他如果不在一开始就记录下来，很可能之后就不好判断。

所以这些，就很适合放进锦囊里。在清醒时写下妙计。

当然我知道，如果这个产品要做的好，后面我还得学习后端。
